!DOCTYPE

Definicion

Se utiliza para especificar la versión de HTML que se está utilizando en la página.

Sintaxis

<!DOCTYPE>

Ejemplo sintaxis

<!DOCTYPE>

img

Definicion

Define una imagen en línea en un documento HTML.

Sintaxis

<img src="ruta_de_la_imagen

Ejemplo sintaxis

<img src="ruta_de_la_imagen

Ejemplo ejecutado

br

Definicion

Define un salto de línea en un documento HTML.

Sintaxis

<br>

Ejemplo sintaxis

<br>

Ejemplo ejecutado


input

Definicion

Define un campo de entrada en un formulario HTML.

Sintaxis

<input type="tipo_de_input" name="nombre_del_input">

Ejemplo sintaxis

<input type="tipo_de_input" name="nombre_del_input">

Ejemplo ejecutado

hr

Definicion

Define una línea horizontal en un documento HTML para separar contenido.

Sintaxis

<hr>

Ejemplo sintaxis

meta

Definicion

Define metadatos sobre el documento HTML, como la codificación de caracteres, la descripción, palabras clave, etc.

Sintaxis

<meta atributo="valor">

Ejemplo sintaxis

<meta atributo="valor">

Ejemplo ejecutado

link

Definicion

Define la relación entre el documento actual y un recurso externo, como una hoja de estilos CSS.

Sintaxis

<link rel="relacion" href="ruta_del_recurso">

Ejemplo sintaxis

<link rel="relacion" href="ruta_del_recurso">

Ejemplo ejecutado

embed

Definicion

Incrusta contenido externo, como imágenes, audio o video, en un documento HTML.

Sintaxis

<embed src="ruta_del_contenido" type="tipo_de_contenido">

Ejemplo sintaxis
