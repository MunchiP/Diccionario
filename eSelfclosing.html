input

Definición: Define un campo de entrada en un formulario HTML.

Sintaxis

<input type="tipo_de_input" name="nombre_del_input">

Ejemplo sintaxis

<input type="tipo_de_input" name="nombre_del_input">

hr

Definición: Define una línea horizontal en un documento HTML para separar contenido.

Sintaxis

<hr>

Ejemplo sintaxis

<hr>

Ejemplo sintaxis

meta

Definición: Define metadatos sobre el documento HTML, como la codificación de caracteres, la descripción, palabras clave, etc.

Sintaxis

<meta atributo="valor">

Ejemplo sintaxis

<meta atributo="valor">

link

Definición: Define la relación entre el documento actual y un recurso externo, como una hoja de estilos CSS.

Sintaxis

<link rel="relacion" href="ruta_del_recurso">

Ejemplo sintaxis

<link rel="relacion" href="ruta_del_recurso">

Ejemplo ejecutado

embed

Definición: Incrusta contenido externo, como imágenes, audio o video, en un documento HTML.

Sintaxis

<embed src="ruta_del_contenido" type="tipo_de_contenido">

Ejemplo sintaxis
