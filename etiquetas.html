a

Definición: La etiqueta a define un enlace hiperactivo, que normalmente se utiliza para vincular una página web a otra.

Sintaxis

<a href="URL_del_destino">Texto del enlace</a>

Ejemplo sintaxis

<a href="https://www.ejemplo.com">Visita nuestro sitio web</a>

Ejemplo ejecutado

Visita nuestro sitio web

article

Definición : Define un contenido independiente y autocontenido en una página web. Por lo general, se utiliza para artículos, blogs, noticias u otra forma de contenido que pueda existir de forma independiente y reutilizable.

Sintaxis

<article>Contenido del artículo</article>

Ejemplo sintaxis

<article>Contenido del artículo</article>

Ejemplo ejecutado

Título del artículo

Este es un párrafo dentro del artículo. El contenido del artículo puede incluir texto, imágenes, videos, etc.

Otro párrafo dentro del artículo.

aside

Definición : Definir contenido secundario o contenido relacionado con el contenido principal de la página.

Sintaxis

<aside>Contenido secundario</aside>

Ejemplo sintaxis

<aside>Contenido secundario</aside>

Ejemplo ejecutado

Contenido relacionado
  • Enlace 1
  • Enlace 2
  • Enlace 3

Sintaxis

<blockquote> Texto citado </blockquote>

<blockquote> Texto citado </blockquote>

Ejemplo ejecutado

cite

Definición: Se utiliza para definir el título de una obra, como un libro, una película, un artículo, etc., que se está citando en un documento HTML.

Sintaxis

Ejemplo sintaxis

div

Definición: La etiqueta div define una división o sección en un documento HTML. Se utiliza como un contenedor genérico para otros elementos.

Sintaxis

<div>Contenido </div>

Ejemplo sintaxis

Ejemplo ejecutado

Este es un párrafo dentro de un div

figure

Definición: Se utiliza para encapsular contenido autónomo, como imágenes, gráficos, ilustraciones, diagramas, código, etc., junto con su leyenda asociada utilizando la etiqueta <figcaption>

Sintaxis

<figure> <figcaption>Descripción del contenido</figcaption> </figure>

Ejemplo sintaxis

<figure> <figcaption>Descripción del contenido</figcaption> </figure>

Ejemplo ejecutado

figcaption

Definición: Se utiliza dentro de un elemento <figure> para proporcionar una leyenda o título descriptivo para la figura.

Sintaxis

<figure> ... </figure>

Ejemplo sintaxis

<figure>
<img src= "ruta de la imagen" alt="descripcion de la imagen">
<figcaption>Leyenda o título descriptivo</figcaption>
</figure>

Ejemplo ejecutado

form

Definición: La etiqueta form define un formulario HTML que permite al usuario ingresar datos que se pueden enviar al servidor para su procesamiento.

Sintaxis

<form action="URL_del_procesador" method="GET/POST">Contenido del formulario</form>

Ejemplo sintaxis

<form action="/procesar_formulario.php" method="post"> <label for="nombre">Nombre:</label> <input type="text" id="nombre" name="nombre"> <input type="submit" value="Enviar">
</form>

Ejemplo ejecutado

mark

Definición: Se utiliza para resaltar partes de un texto, indicando que han sido marcadas o resaltadas por algún motivo en particular, como resultados de búsqueda, términos clave, cambios recientes, etc.

Sintaxis

<mark> ... </mark>

Ejemplo sintaxis

<mark>Texto resaltado</mark>

Ejemplo ejecutado

El texto resaltado es útil para enfatizar partes importantes de un documento.

p

Definición: La etiqueta p define un párrafo de texto.

Sintaxis

<p>Este es un párrafo de ejemplo.</p>

Ejemplo sintaxis

<p> Lorem ipsum dolor sit amet consectetur, adipisicing elit. Quod, quae!.</p>

Ejemplo ejecutado

Lorem ipsum dolor, sit amet consectetur adipisicing elit. Id, maiores.

pre

Definición: Se utiliza para representar texto preformateado, lo que significa que el texto dentro de esta etiqueta se muestra tal como está escrito en el código fuente, respetando espacios en blanco, saltos de línea y cualquier otro formato de texto.

Sintaxis

<pre> ... </pre>

Ejemplo sintaxis

<pre> ... </pre>

Ejemplo ejecutado

<pre> hola mundo <p>Hola Mundo</p> </pre>

ul, ol, li

Definición: ul crea una lista no ordenada, ol crea una lista ordenada, y li define elementos de lista en ambos tipos de listas.

Sintaxis

<ul> #li dentro </ul> <ol> #li dentro </ol> <li>Manzanas</li>

Ejemplo sintaxis

<ul>; <li>Manzanas</li> <li>Plátanos</li> </ul> <ol> <li>Primer paso</li> <li>Segundo paso</li> </ol>

Ejemplo ejecutado

  • Manzanas
  • Plátanos
  1. Primer paso
  2. Segundo paso

sub / sup

Definición:
sub = se utiliza para representar texto subíndice.
sup = se usa para representar texto sobrescrito.

Sintaxis

<sub > ... </sub>

<sup > ... </sup>

Ejemplo sintaxis

<sub>Texto subíndice</sub>

<sup>Texto sobrescrito</sup>

Ejemplo ejecutado

La fórmula química del agua es H2O.

El teorema de Pitágoras se expresa como a2 + b2 = c2.

span

Definición: La etiqueta span se utiliza para aplicar estilo o comportamiento a una parte específica del texto dentro de un elemento en línea.

Sintaxis

<span style="color: red;">texto con estilo</span>

Ejemplo sintaxis

<p>Este es un <span style="color: red;">texto con estilo</span>.</p>

Ejemplo ejecutado

Este es un texto con estilo.

Ejemplo sintaxis

Ejemplo ejecutado

style

Definición: Se utiliza para definir estilos CSS dentro del documento HTML. Puedes usarla para incluir estilos en línea directamente en el HTML o para enlazar a un archivo externo de hojas de estilo CSS.

Sintaxis

Ejemplo sintaxis
