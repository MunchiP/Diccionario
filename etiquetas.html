a

Definicion

La etiqueta a define un enlace hiperactivo, que normalmente se utiliza para vincular una página web a otra.

Sintaxis

<a href="URL_del_destino">Texto del enlace</a>

Ejemplo sintaxis

<a href="https://www.ejemplo.com">Visita nuestro sitio web</a>

Ejemplo ejecutado

Visita nuestro sitio web

b

Definicion

Esta etiqueta dará formato de negrita a cualquier texto que se encuentre entre la etiqueta.

Sintaxis

<b>....</b>

Ejemplo sintaxis

<b>....</b>

Ejemplo ejecutado

<b>Este texto esta en negrilla</b>

body

Definicion

Define la sección del cuerpo de la página web, donde se incluyen todos los elementos que se mostrarán en la página.

Sintaxis

<body>....</body>

Ejemplo sintaxis

<body>....</body>

div

Definicion

La etiqueta div define una división o sección en un documento HTML. Se utiliza como un contenedor genérico para otros elementos.

Sintaxis

<div>Contenido </div>

Ejemplo sintaxis

<div>
<p>Este es un párrafo dentro de un div.</p>
</div>

Ejemplo ejecutado

Este es un párrafo dentro de un div

form

Definicion

La etiqueta form define un formulario HTML que permite al usuario ingresar datos que se pueden enviar al servidor para su procesamiento.

Sintaxis

<form action="URL_del_procesador" method="GET/POST">Contenido del formulario</form>

Ejemplo sintaxis

<form action="/procesar_formulario.php" method="post">
<label for="nombre">Nombre:</label>
<input type="text" id="nombre" name="nombre">
<input type="submit" value="Enviar">
</form>

Ejemplo ejecutado


h1 - h6

Sintaxis

<h4>Texto del encabezado</h4>

<h6>Texto del encabezado</h6>

p

Definicion

La etiqueta p define un párrafo de texto.

Sintaxis

<p>Este es un párrafo de ejemplo.</p>

Ejemplo sintaxis

<p> Lorem ipsum dolor sit amet consectetur, adipisicing elit. Quod, quae!.</p>

Ejemplo ejecutado

Lorem ipsum dolor, sit amet consectetur adipisicing elit. Id, maiores.

Sintaxis

Ejemplo sintaxis

Ejemplo ejecutado

ul, ol, li

Definicion

ul crea una lista no ordenada, ol crea una lista ordenada, y li define elementos de lista en ambos tipos de listas.

Sintaxis

<ul> #li dentro </ul> <ol> #li dentro </ol> <li>Manzanas</li>

Ejemplo sintaxis

<ul> <li>Manzanas</li> <li>Plátanos</li> </ul> <ol> <li>Primer paso</li> <li>Segundo paso</li> </ol>

Ejemplo ejecutado

  • Manzanas
  • Plátanos
  1. Primer paso
  2. Segundo paso

span

Definicion

La etiqueta span se utiliza para aplicar estilo o comportamiento a una parte específica del texto dentro de un elemento en línea.

Sintaxis

<p>Este es un <span style="color: red;">texto con estilo</span>.</p>

Ejemplo sintaxis

<p>Este es un <span style="color: red;">texto con estilo</span>.</p>

Ejemplo ejecutado

Este es un texto con estilo.

Strong

Definicion

se utiliza para definir texto que se considera de importancia semántica fuerte o enfática, generalmente indicando que su contenido debe ser mostrado en negrita por los navegadores web.

Sintaxis

<strong> ... </strong>

Ejemplo sintaxis

<strong> ... </strong>

Ejemplo ejecutado

Letra en negrilla

table

Definicion

La etiqueta table define una tabla HTML.

Sintaxis

Lorem ipsum dolor sit amet.

Ejemplo

<table>
<tr>
<th>Encabezado 1</th>
<th>Encabezado 2</th> </tr>
<tr>
<td>Dato 1</td>
<td>Dato 2</td>
</tr>
</table>

Encabezado 1 Encabezado 2
Dato 1 Dato 2
